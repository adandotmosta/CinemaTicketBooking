2. Create a PostgreSQL Database and User:
bash
Copy code
sudo -u postgres psql

# Inside the PostgreSQL shell, create a database and user:
CREATE DATABASE your_database_name;
CREATE USER your_username WITH PASSWORD 'your_password';

# Grant privileges to the user on the database:
ALTER ROLE your_username SET client_encoding TO 'utf8';
ALTER ROLE your_username SET default_transaction_isolation TO 'read committed';
ALTER ROLE your_username SET timezone TO 'UTC';
GRANT ALL PRIVILEGES ON DATABASE your_database_name TO your_username;

# Exit the PostgreSQL shell:
\q
3. Update Django Settings:
In your Django project's settings.py file, update the DATABASES configuration to use PostgreSQL:

python
Copy code
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.postgresql',
        'NAME': 'your_database_name',
        'USER': 'your_username',
        'PASSWORD': 'your_password',
        'HOST': 'localhost',
        'PORT': '5432',  # Default PostgreSQL port
    }
}
Replace your_database_name, your_username, and your_password with the values you set during the PostgreSQL setup.

4. Apply Migrations:
After updating the settings, apply the migrations to create the necessary tables in the PostgreSQL database:

bash
Copy code
python manage.py makemigrations
python manage.py migrate
